# 宇晶味美株式会社 - Yujing Weimei Co., Ltd.

日本商品全球贸易平台

## 项目简介

宇晶味美株式会社是一家专业的日本商品全球贸易公司，致力于将优质的日本产品带到世界各地。

## 功能特性

- 🏠 现代化首页展示
- 🛍️ 产品介绍与展示
- 👥 团队介绍
- 📞 联系方式
- 📱 响应式设计
- 🎨 欧美简约风格

## 使用方法

1. 克隆仓库
   ```bash
   git clone https://github.com/sennukann/yujing-weimei-website.git
   ```

2. 在本地打开 `index.html`
   ```bash
   open index.html
   # 或在浏览器中打开文件
   ```

3. 或部署到 GitHub Pages
   - 在仓库设置中启用 GitHub Pages
   - 选择 main 分支作为源
   - 访问 `https://sennukann.github.io/yujing-weimei-website/`

## 自定义指南

### 修改公司信息
1. 编辑所有 `.html` 文件中的公司名称
2. 更新 `contact.html` 中的地址、电话、邮箱
3. 修改 `team.html` 中的团队成员信息

### 修改样式
1. 主颜色在 `css/style.css` 中的 `.hero` 部分
2. 字体可在 `body` 选择器中修改
3. 间距和布局在响应式部分调整

### 添加图片
1. 创建 `images/` 文件夹
2. 替换 `.product-image` 和 `.member-avatar` 中的 emoji
3. 使用 `<img>` 标签替代 emoji
